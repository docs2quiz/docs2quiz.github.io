---
title: Kuis Pertamamu
description: Panduan 5 menit dari unggah dokumen sampai kuis siap ekspor.
---

Panduan ini menjalankan seluruh alurnya sekali, dari awal sampai akhir.

## 1. Buat knowledge base

Di halaman utama, klik **+ Create Knowledge**. Beri judul dan pilih satu atau lebih file (`.pdf`, `.docx`, `.txt`, `.md`).

Knowledge base akan mulai dengan status **Embedding Processing** selagi dokumenmu dipotong dan diindeks. Ini bisa memakan waktu beberapa detik sampai beberapa menit tergantung ukuran dokumen — halaman akan otomatis polling dan update begitu selesai.

## 2. Tunggu sampai "Ready"

Setelah badge status berubah jadi **Ready**, klik kartunya untuk membuka.

## 3. Masukkan kata kunci

Di halaman generate kamu akan melihat file sumber dan berapa banyak chunk yang terindeks. Ketik kata kunci atau topik di kolom **Keyword** — sesuatu yang kamu tahu ada di dokumenmu. Belum yakin isi dokumennya apa? Klik **📊 Keyword Stats** untuk melihat istilah paling sering muncul di kontenmu.

## 4. Generate

Klik **Generate Questions**. Soal muncul satu per satu saat dihasilkan dan diverifikasi — kamu akan melihat counter progres langsung. Setiap soal yang selesai menampilkan opsinya, dan blok **Source Evidence** yang mengutip persis dokumen dan potongan teks mana ia berdasar.

## 5. Ekspor

Setelah ada hasil, gunakan menu **⬇ Export** untuk unduh sebagai PDF (siap cetak), DOCX (bisa diedit), atau JSON (data mentah).

Selesai — lihat [Membuat Soal](/id/guides/generating-questions/) untuk semua yang bisa dilakukan halaman generate, atau [Knowledge Base](/id/guides/knowledge-bases/) untuk detail mengelola konten yang diunggah.
